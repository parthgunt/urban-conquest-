# Urban Conquest

Urban Conquest is a hackathon project built as an experimental, interactive,
territory-based web application.

## Overview
The project explores map-driven interactions and territory mechanics using a
frontend-focused approach.

## Tech Stack
- HTML
- CSS
- JavaScript
